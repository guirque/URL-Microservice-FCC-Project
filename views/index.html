What is the URL Shortener?

It's an API that is meant to help, as the name implies, shortening URLs, in order to provide users the ability to share links to their favorite websites without the need for long and confusing text.

How Can I Use it?

Post the URL in the form in the above section, and you will receive an object containing a "short_url" key. It holds the number of your new link. That means that accessing this website /api/shorturl/[your number] will redirect to the URL you posted.
